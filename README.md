# 墨境棋局 - 水墨风卡牌Roguelike游戏

## 项目概述
一款基于Phaser 3引擎的水墨风格卡牌Roguelike游戏，融合HD-2D美术风格与策略卡牌玩法。

## 游戏特色
- **3章节完整剧情**：墨林迷踪 → 影武者殿堂 → 墨渊深处
- **丰富的卡牌系统**：12种基础卡牌，支持升级和组合
- **策略性战斗**：AP系统、连击机制、状态效果
- **Roguelike元素**：随机地图、事件、商店、休息点
- **HD-2D美术方案**：完整的美术规范文档

## 快速开始

### 运行Demo
直接用浏览器打开 `dist/index.html` 即可开始游戏。

```bash
# 或者启动本地服务器
cd dist
python -m http.server 8080
# 访问 http://localhost:8080
```

### 开发模式
```bash
# 安装依赖
npm install

# 开发模式（热重载）
npm run dev

# 构建生产版本
npm run build

# 类型检查
npm run typecheck

# 运行测试
npm run test
```

## 游戏流程
1. **主菜单** → 开始游戏
2. **地图探索** → 选择节点前进
3. **战斗系统** → 出牌、连击、状态效果
4. **奖励选择** → 获得新卡牌
5. **章节过渡** → 击败BOSS进入下一章
6. **通关** → 完成3章剧情

## 核心系统

### 战斗机制
- **AP系统**：每回合3点行动力（可升级至5点）
- **卡牌类型**：攻击牌、技能牌、能力牌
- **连击系统**：连续出攻击牌+2伤害
- **状态效果**：易伤、虚弱、中毒、力量、再生

### 卡牌系统
- **初始牌组**：5张斩击 + 4张防御 + 1张快斩
- **卡牌升级**：在休息点可升级卡牌
- **奖励机制**：战斗胜利后3选1

### 地图系统
- **节点类型**：战斗(60%)、精英(15%)、事件(10%)、商店(8%)、休息(7%)
- **章节进度**：每章16个节点 + 1个BOSS
- **难度递增**：敌人属性随章节提升

### 存档系统
- 自动保存到localStorage
- 支持暂停和继续游戏
