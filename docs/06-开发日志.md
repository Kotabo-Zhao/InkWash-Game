## 2026-07-19 - v0.2.2-alpha (美术管线v2: 分层控制+版本管理)

### 修改
- [管线] art_pipeline.py 重写为v2
  - 新增VersionManager：每次生成记录所有参数（模型版本、seed、LoRA、ControlNet参数）
  - 新增PromptBuilder：构建包含IP-Adapter和ControlNet的完整工作流
  - 新增角色参考图生成（一次性锁定外观）
  - 新增ControlNet OpenPose集成（控制动作姿态）
  - 新增IP-Adapter集成（保持角色一致性）
  - 新增精灵表自动打包
  - 新增去背景后处理
  - 新增`--versions`命令查看历史版本
- [文档] 04-美术规范.md 升级到v2
  - 新增"分层控制策略"章节
  - VNCCS POSE STUDIO动作帧方案
  - 版本管理说明
  - ComfyUI工作流文件说明
  - 资源目录结构更新（新增refs/versions目录）

### 核心设计决策
- 纯Prompt无法保证角色一致性 → IP-Adapter + 参考图
- 纯文本无法控制动作连贯性 → ControlNet OpenPose + VNCCS POSE STUDIO
- 无法复现问题 → VersionManager记录所有参数
- 风格飘移 → 风格LoRA + 固定negative prompt

### 管线使用方式
```bash
# 生成角色参考图（一次性）
python scripts/art_pipeline.py --character protagonist

# 生成所有资源
python scripts/art_pipeline.py --all

# 查看版本历史
python scripts/art_pipeline.py --versions
```

### 下一步
1. 准备ComfyUI工作流JSON文件（character_ref.json, character_frame.json）
2. 从免费素材网站下载参考动作帧
3. VNCCS POSE STUDIO提取骨骼姿态
4. 测试完整生成流程
