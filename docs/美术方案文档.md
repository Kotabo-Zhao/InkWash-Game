# 墨境棋局 - HD-2D美术方案

## 美术风格定位

**HD-2D**：结合像素艺术与现代光影效果的视觉风格，参考《八方旅人》《三角战略》等作品。

### 核心视觉元素
- **像素基础**：16×16 / 32×32 像素网格
- **水墨滤镜**：整体色调偏冷，模拟水墨画的淡雅质感
- **光影效果**：动态光照、粒子特效、体积雾
- **场景分层**：背景层 + 中景层 + 角色层 + 前景层

---

## 角色精灵表规格

### 基础规格
- **尺寸**：128×128px（含8方向）
- **方向**：8方向（N, NE, E, SE, S, SW, W, NW）
- **动画帧数**：
  - idle：4帧（循环）
  - walk：6帧（循环）
  - attack：4帧（单次）
  - run：6帧（循环，速度1.5x）
  - dead：4帧（单次）

### 文件命名规范
```
assets/sprites/{character}/{character}_{direction}_{action}_{frame}.png
```

**示例**：
```
assets/sprites/protagonist/protagonist_n_idle_0.png
assets/sprites/protagonist/protagonist_ne_walk_2.png
assets/sprites/protagonist/protagonist_e_attack_1.png
```

### 精灵表（Sprite Sheet）格式
```
assets/sprites/{character}/{character}_spritesheet.png
```

**布局**：
- 横向：动作类型（idle, walk, attack, run, dead）
- 纵向：方向（N, NE, E, SE, S, SW, W, NW）
- 每帧128×128px

## 场景背景规格

### 基础规格
- **分辨率**：800×600px（可缩放）
- **格式**：PNG（透明通道）
- **分层**：
  - `background_far.png`：远景（模糊，视差滚动慢）
  - `background_mid.png`：中景（清晰，视差滚动中）
  - `background_near.png`：近景（清晰，视差滚动快）
  - `foreground.png`：前景遮挡（透明PNG）

### 文件结构
```
assets/backgrounds/{chapter}/{layer}.png
```

**示例**：
```
assets/backgrounds/chapter1/background_far.png
assets/backgrounds/chapter1/background_mid.png
assets/backgrounds/chapter1/background_near.png
assets/backgrounds/chapter1/foreground.png
```

## 特效系统

### 粒子特效

#### 攻击特效
- **斩击**：白色刀光轨迹，持续0.3s
- **重斩**：金色刀光，带震动效果
- **横扫**：扇形范围，多层刀光

#### 防御特效
- **护甲**：金色护盾光圈，环绕角色
- **铁布衫**：厚实的金色护盾，带金属质感

#### 法术特效
- **冥想**：紫色光环，向上升腾
- **治愈之光**：绿色光点，从天而降

#### 环境特效
- **墨雾**：半透明黑色雾气，缓慢飘动
- **光晕**：柔和的黄色光晕，点缀场景
- **墨滴**：从上方滴落的墨滴，碰到地面扩散

### 后处理效果

#### 水墨滤镜
```glsl
// 伪代码示例
uniform sampler2D u_texture;
uniform float u_inkDensity; // 水墨浓度

void main() {
    vec4 color = texture2D(u_texture, v_texCoord);
    
    // 降低饱和度
    float gray = dot(color.rgb, vec3(0.299, 0.587, 0.114));
    vec3 desaturated = mix(color.rgb, vec3(gray), 0.3);
    
    // 添加水墨质感
    float noise = fract(sin(dot(v_texCoord, vec2(12.9898, 78.233))) * 43758.5453);
    vec3 finalColor = mix(desaturated, vec3(0.1, 0.1, 0.15), noise * u_inkDensity);
    
    gl_FragColor = vec4(finalColor, color.a);
}
```

#### 光影效果
- **动态光照**：角色周围柔和光圈
- **体积雾**：场景底部半透明雾气
- **景深模糊**：远景轻微模糊（可选）

## 角色生成流程（ComfyUI + VNCCS POSE STUDIO）

> **已废弃** - 请参考 `docs/04-美术规范.md` 获取最新方案。
> VNCCS POSE STUDIO 是免费工具，用于将参考动作图转换成目标角色姿势。

### 工具链
- **ComfyUI**：基础生成（IP-Adapter + ControlNet + 风格LoRA）
- **VNCCS POSE STUDIO**：免费工具，将参考图转换成目标姿势（无需授权）
- **Stable Diffusion**：底层模型

### 生成步骤（简化版）

```
1. ComfyUI生成角色参考图（三视图）→ 锁定外观
2. 从免费素材网站下载参考动作帧
3. VNCCS POSE STUDIO：参考动作图 → 转换为我们角色的姿势
4. ComfyUI：角色参考图(IP-Adapter) + 姿态图(ControlNet) → 最终精灵帧
5. 后处理：resize + 去背景 + 打包精灵表
```

详细流程见 `docs/04-美术规范.md` 第六章。

## 场景生成流程

### 工具链
- **Midjourney / DALL-E 3**：基础场景生成
- **Photoshop**：分层处理
- **Aseprite**：像素化处理

### 生成步骤

#### 1. 生成基础场景
```
Prompt: "HD-2D pixel art ink wash style, ancient Chinese forest, 
mystical atmosphere, ink trees, paper ground, 
distant mountains, soft lighting, 
--ar 4:3 --v 5"
```

#### 2. 分层处理
```bash
# 使用Photoshop脚本自动分层
python split_layers.py \
  --input scene_base.png \
  --output_layers far,mid,near,foreground \
  --blur_far 5 \
  --blur_mid 2
```

#### 3. 像素化处理
```bash
# 使用Aseprite批量像素化
for layer in far mid near foreground; do
  aseprite --batch \
    --scale 0.25 \
    --output "background_${layer}_pixel.png" \
    "background_${layer}.png"
done
```

#### 4. 导出最终资源
```bash
# 调整尺寸并导出
python export_backgrounds.py \
  --input_dir assets/backgrounds/raw/ \
  --output_dir assets/backgrounds/ \
  --resolution 800x600
```

## 性能优化建议

### 精灵表优化
- **合并小图**：将多个小图标合并到一张大图
- **使用纹理图集**：TexturePacker生成图集
- **压缩格式**：使用WebP（如有支持）

### 背景优化
- **视差滚动**：只更新可见层
- **LOD系统**：远处背景降低分辨率
- **缓存机制**：预渲染静态层

### 特效优化
- **对象池**：粒子使用对象池复用
- **帧预算**：控制每帧特效数量（目标30FPS）
- **降级策略**：低性能设备减少粒子数
