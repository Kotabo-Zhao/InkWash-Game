# 墨境棋局 - 世界观与角色设定

## 世界观背景

在遥远的东方，有一片被称为"墨境"的神秘领域。这里并非人间，也非地狱，而是由古老的墨灵创造的世界——一个由笔墨纸砚构成的维度。

千年前，墨境的守护者"墨龙"陷入了沉睡，墨境开始崩坏。原本平和的墨灵变成了凶猛的墨兽，四处游荡。曾经美丽的墨林变成了阴暗的迷阵，曾经庄严的殿堂被影武者占据，曾经清澈的墨渊变成了深不见底的黑暗。

你是被选中的"棋手"，肩负着重新唤醒墨龙的使命。你必须穿越墨林、击败影武者、深入墨渊，最终面对墨渊守护者——那个导致墨龙沉睡的元凶。

你的武器是"卡牌"，每一张都蕴含着墨境的力量。斩击、防御、法术——这些卡牌将在战斗中助你一臂之力。但记住，墨境的力量是双刃剑，使用不当也会反噬自身。

墨境的未来，就在你的手中。

## 主要角色设定

### 1. 棋手（主角）
- **身份**：被选中的勇者，肩负唤醒墨龙的使命
- **性格**：坚毅、智慧过人
- **动机**：拯救墨境，唤醒墨龙，恢复世界的和平
- **能力**：使用卡牌战斗，掌握斩击、防御、法术等多种技能

### 2. 墨灵（引导NPC）
- **身份**：墨境的原住民，形态如流动的墨水
- **性格**：神秘、友善
- **动机**：帮助棋手完成使命，恢复墨境的和平
- **作用**：在旅途中给予棋手指引和帮助，提供休息点和商店服务

### 3. 影武者（敌对NPC）
- **身份**：曾经是高贵的墨境守护者，如今被黑暗侵蚀
- **性格**：扭曲、执着
- **动机**：阻止棋手前进，维护自己扭曲的"秩序"
- **能力**：掌握强大的影系技能，是棋手的强敌

### 4. 行商（商店NPC）
- **身份**：在墨境中旅行的神秘商人
- **性格**：圆滑、神秘
- **动机**：在墨境的混乱中谋利，同时也帮助棋手
- **作用**：贩卖各种奇珍异宝，无人知晓他的真实身份

### 5. 墨龙王（最终BOSS）
- **身份**：墨境的创造者与守护者，拥有无穷的力量
- **性格**：威严、沉睡
- **动机**：沉睡中等待棋手的到来，测试棋手是否配得上唤醒自己
- **能力**：掌控墨境的核心力量，是最终考验

## 章节设定

### 第一章：墨林迷踪
- **场景描述**：墨境入口处的古老森林，充满了初级墨兽
- **背景氛围**：树木由墨水构成，地面是湿润的宣纸，空气中弥漫着墨香
- **主要敌人**：墨兽、墨狼
- **BOSS**：墨龙王（第一章版本，HP: 80）

### 第二章：影武者殿堂
- **场景描述**：曾经的圣殿，如今被影武者占据
- **背景氛围**：宏伟的建筑已显破败，墙壁上残留着古老的壁画，影武者在其中游荡
- **主要敌人**：影刺客、影武者
- **BOSS**：影武者（HP: 100）

### 第三章：墨渊深处
- **场景描述**：墨境的最深处，墨龙王沉睡之地
- **背景氛围**：黑暗的深渊，只有微弱的墨光指引方向，强大的墨元素在守护着最后的秘密
- **主要敌人**：墨元素
- **BOSS**：墨渊守护者（HP: 120）

## 数值设计原则

### 卡牌平衡
- **攻击牌**：伤害范围 6-15
- **技能牌**：护甲范围 3-12
- **费用**：0-2 AP

### 敌人属性
- **普通敌人**：HP 20-45，伤害 6-10
- **精英敌人**：HP 45-60，伤害 10-12
- **BOSS**：HP 80-120，伤害 12-18

### 成长曲线
- **章节升级**：每章完成 HP+10，AP+1（最大5）
- **初始属性**：HP 60，AP 3
- **初始牌组**：5斩击 + 4防御 + 1快斩 = 10张

### 奖励机制
- **战斗胜利**：获得金币 + 3选1卡牌
- **卡池权重**：普通卡权重高，稀有卡权重低
- **章节解锁**：第二章解锁史诗卡，第三章卡池偏向高级卡

## 玩法系统

### 回合制流程
1. **玩家回合**：出牌、消耗AP、结束回合
2. **敌人回合**：执行意图（攻击/防御/增益）
3. **结算阶段**：检查胜负、抽牌、进入下一回合

### AP系统
- **初始AP**：3点
- **每回合重置**：回合开始时AP恢复至最大值
- **最大AP**：5点（通过章节升级）

### 状态系统
- **易伤**（debuff）：受到伤害+50%，持续2回合
- **虚弱**（debuff）：造成伤害-30%，持续2回合
- **中毒**（debuff）：每回合受到等层数伤害，永久
- **力量**（buff）：每层+1伤害，永久
- **再生**（buff）：每回合恢复等层数HP，持续3回合

### 连击系统
- **机制**：连续出攻击牌时，每张+2伤害
- **上限**：无限制，但AP消耗会限制连击次数

### 升级机制
- **经验获取**：战斗胜利获得经验值
- **升级效果**：每级可选择一张新卡牌加入牌组
- **升级曲线**：每级所需经验 = 当前等级 × 50
