# 墨境棋局 - 世界观设定

> 本文档定义游戏的世界观、角色、剧情和对话文本。

---

## 一、世界观背景

### 墨境的起源

在遥远的东方，有一片被称为"墨境"的神秘领域。这里并非人间，也非地狱，而是由古老的墨灵创造的世界——一个由笔墨纸砚构成的维度。

墨境的核心是"墨龙"，它是墨境的创造者与守护者。墨龙沉眠于墨渊深处，它的呼吸化作墨雾，它的血液化作墨河，它的鳞片化作墨石。墨境的一切生命都源于墨龙的力量。

### 墨境的崩坏

千年前，墨龙陷入了沉睡。没有人知道原因，只知道从那一刻起，墨境开始崩坏。

原本平和的墨灵变成了凶猛的墨兽，四处游荡。曾经美丽的墨林变成了阴暗的迷阵，曾经庄严的殿堂被影武者占据，曾经清澈的墨渊变成了深不见底的黑暗。

墨境的秩序正在崩溃，而这一切的根源，似乎与墨渊深处的"守护者"有关——那个导致墨龙沉睡的元凶。

### 棋手的使命

你是被选中的"棋手"，肩负着重新唤醒墨龙的使命。

你必须穿越墨林、击败影武者、深入墨渊，最终面对墨渊守护者。你的武器是"卡牌"，每一张都蕴含着墨境的力量。斩击、防御、法术——这些卡牌将在战斗中助你一臂之力。

但记住，墨境的力量是双刃剑，使用不当也会反噬自身。

墨境的未来，就在你的手中。

## 二、主要角色

### 1. 棋手（主角）

- **身份**：被选中的勇者，肩负唤醒墨龙的使命
- **性格**：坚毅、智慧过人
- **动机**：拯救墨境，唤醒墨龙，恢复世界的和平
- **能力**：
  - 使用卡牌战斗（斩击、防御、法术）
  - 掌握棋盘落子之术（5×5棋盘策略）
  - 操控墨压释放终极技
  - 组合阵型获得加成

### 2. 墨灵（引导NPC）

- **身份**：墨境的原住民，形态如流动的墨水
- **性格**：神秘、友善
- **动机**：帮助棋手完成使命，恢复墨境的和平
- **作用**：
  - 在旅途中给予棋手指引和帮助
  - 提供休息点和商店服务
  - 在关键时刻给予提示

### 3. 影武者（敌对NPC）

- **身份**：曾经是高贵的墨境守护者，如今被黑暗侵蚀
- **性格**：扭曲、执着
- **动机**：阻止棋手前进，维护自己扭曲的"秩序"
- **能力**：
  - 掌握强大的影系技能
  - 是棋手的强敌（第二章BOSS）
  - 会落子对抗玩家

### 4. 行商（商店NPC）

- **身份**：在墨境中旅行的神秘商人
- **性格**：圆滑、神秘
- **动机**：在墨境的混乱中谋利，同时也帮助棋手
- **作用**：
  - 贩卖各种奇珍异宝
  - 无人知晓他的真实身份

### 5. 墨龙王（最终BOSS）

- **身份**：墨境的创造者与守护者，拥有无穷的力量
- **性格**：威严、沉睡
- **动机**：沉睡中等待棋手的到来，测试棋手是否配得上唤醒自己
- **能力**：
  - 掌控墨境的核心力量
  - 是最终考验（第三章BOSS）

## 三、章节设定

### 第一章：墨林迷踪

- **场景描述**：墨境入口处的古老森林，充满了初级墨兽
- **背景氛围**：树木由墨水构成，地面是湿润的宣纸，空气中弥漫着墨香
- **主要敌人**：墨兽、墨狼
- **BOSS**：墨龙王（第一章版本，HP: 80）
- **章节特色**：
  - 入门级难度
  - 学习棋盘落子基础
  - 掌握位置加成

### 第二章：影武者殿堂

- **场景描述**：曾经的圣殿，如今被影武者占据
- **背景氛围**：宏伟的建筑已显破败，墙壁上残留着古老的壁画，影武者在其中游荡
- **主要敌人**：影刺客、影武者
- **BOSS**：影武者（HP: 100）
- **章节特色**：
  - 中级难度
  - 敌人AI更聪明
  - 学习阵型组合

### 第三章：墨渊深处

- **场景描述**：墨境的最深处，墨龙王沉睡之地
- **背景氛围**：黑暗的深渊，只有微弱的墨光指引方向，强大的墨元素在守护着最后的秘密
- **主要敌人**：墨元素
- **BOSS**：墨渊守护者（HP: 120）
- **章节特色**：
  - 高级难度
  - 墨压管理至关重要
  - 终极挑战

## 五、核心机制与世界观的融合

### 棋盘落子的意义

在墨境中，每一次战斗都是在棋盘上的博弈。落子不仅仅是策略，更是对墨境力量的操控。

- **天元**：墨境的核心，力量最为集中，因此伤害翻倍
- **四角**：墨境的边界，防御最为稳固，因此护甲增加
- **边缘**：墨境的脉络，连接各方，因此抽牌增加

### 墨压的含义

墨压代表棋手对墨境力量的掌控程度。当墨压充满时，棋手可以释放终极技"墨染乾坤"——这是墨境最强大的力量，能够清空棋盘，重创所有敌人。

### 阵型的象征

阵型是棋手与墨境力量共鸣的体现。当特定的位置组合形成时，会产生强大的加成效果：

- **四角阵**：掌控四方，全屏震慑
- **十字阵**：贯通天地，力量倍增
- **一线阵**：脉络畅通，资源源源不断
- **对角阵**：阴阳平衡，防御坚固
- **墨环阵**：核心稳固，墨压激增
