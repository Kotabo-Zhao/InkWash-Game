3.2 子场景复用策略

子场景 = 章节背景 + 遮罩/叠加层
商店: 章节背景 + 灯笼/摊位前景遮罩 → 复用bgAsset
休息点: 章节背景 + 篝火/帐篷遮罩 → 复用bgAsset
事件: 章节背景 + 迷雾遮罩 → 复用bgAsset
宝箱: 章节背景 + 光柱特效 → 复用bgAsset

结论: 每章只需生成4张战斗背景图层, 子场景通过代码叠加实现, 不额外生成。

9.1 角色/敌人精灵

验收项标准检测方法
背景纯净度边缘像素 >95% 为纯白/透明rembg去背后自动检测
角色完整性全身可见, 宽高比 0.4-1.2sprite_selector.py 凸包检测

边缘清晰度Sobel梯度 >阈值自动检测
文字/伪影无可见文字/水印顶部水平结构检测

验收项标准
尺寸800×600
图层分离各层独立, 可单独视差滚动

无伪影无接缝/拉伸/重复纹理

尺寸240×320 px

风格统一所有卡牌色调/笔触一致
稀有度区分边框颜色正确 (白/蓝/紫/金)
